# 4 分钟演示视频提纲

文件命名建议：`组号-演示.mp4`

## 0:00-0:20 开场

大家好，我们展示的是基于 Qt/C++ 实现的记账本系统。它支持用户登录、账本管理、收支记录、预算提醒、统计分析和多人账本邀请加入。

## 0:20-0:45 登录与用户数据隔离

展示登录界面，使用一个账号登录。说明不同用户登录后，账本、记录、预算、标签和个人信息会分别保存在独立用户目录中，避免数据混用。

## 0:45-1:20 首页概览

进入主页，展示本月预算、预算提示、本月结余、本月总收入、本月总支出和记账笔数。说明首页会根据当前账本和当前月份自动汇总。

## 1:20-1:55 添加收支记录

点击“添加收支记录”，选择账本、日期、收入或支出、分类、金额和备注，可选上传图片凭证。保存后回到主页，展示金额和笔数发生变化。

## 1:55-2:25 明细管理

进入明细页，展示当前账本记录列表。演示按分类筛选、修改记录、删除记录和查看图片凭证。说明分类来自用户自定义标签。

## 2:25-2:55 预算管理

进入预算页，选择月份，输入预算金额并保存。展示预算金额、已使用、剩余和进度条。说明预算提示会根据支出情况更新。

## 2:55-3:20 统计分析

进入统计页，展示收入分类饼图、支出分类饼图和历史月度收支趋势折线图。说明图表会读取当前账本数据并自动统计。

## 3:20-3:50 账本管理和多人账本

进入账本管理，展示创建账本、切换当前账本、查看账本详情、修改和删除账本。演示多人账本入口，说明新建账本会直接创建为可邀请账本，系统生成邀请码，其他用户输入邀请码即可加入。

## 3:50-4:00 总结

总结：本系统完成了本地记账、多用户隔离、预算提醒、统计分析和联网多人账本功能，覆盖了 Qt 界面开发、文件读写、JSON 数据处理和 TCP 网络通信。
